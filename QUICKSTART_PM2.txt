================================================================================
              KASIOON BOT - QUICK START WITH PM2
================================================================================

Fast setup guide for running Kasioon Bot with PM2.

================================================================================
STEP 1: INSTALL PM2 (if not installed)
================================================================================

npm install -g pm2

================================================================================
STEP 2: INSTALL DEPENDENCIES
================================================================================

cd /var/www/html/kasioon-bot
npm install

================================================================================
STEP 3: CONFIGURE ENVIRONMENT
================================================================================

cp env-config.txt .env
nano .env

Update with your actual credentials:
- TELEGRAM_BOT_TOKEN (already set in your env-config.txt)
- OPENAI_API_KEY or ANTHROPIC_API_KEY
- KASIOON_API_URL (when available)
- KASIOON_API_KEY (when available)

================================================================================
STEP 4: START THE SERVER
================================================================================

./pm2-start.sh

This will:
✓ Check/install PM2
✓ Create necessary directories
✓ Start kasioon-bot process
✓ Setup auto-startup on reboot
✓ Show status

================================================================================
STEP 5: VERIFY IT'S RUNNING
================================================================================

# Check status
pm2 status

# Test API
curl http://localhost:3355/health

# View logs
pm2 logs kasioon-bot

================================================================================
USEFUL COMMANDS
================================================================================

Start:     ./pm2-start.sh
Stop:      ./pm2-stop.sh
Restart:   ./pm2-restart.sh
Logs:      pm2 logs kasioon-bot
Status:    pm2 status
Monitor:   pm2 monit

================================================================================
AUTO-START ON REBOOT
================================================================================

PM2 startup is configured automatically by pm2-start.sh.

To manually setup:
pm2 startup systemd
pm2 save

================================================================================
NEXT STEPS
================================================================================

1. Configure Telegram webhook:
   curl -X POST "https://api.telegram.org/bot<YOUR_TOKEN>/setWebhook" \
     -d "url=http://chato-app.com/api/telegram/webhook"

2. Test your bot on Telegram

3. Complete kasioon.com API integration (see TODO.txt)

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Port 3355 already in use
Solution: 
   netstat -tulpn | grep 3355
   # Kill the process or change PORT in .env

Problem: PM2 not found
Solution:
   npm install -g pm2

Problem: App won't start
Solution:
   pm2 logs kasioon-bot --err
   # Check .env file configuration
